### What I learned

First of all, I learned fundamentals of React and Tailwind. I learned how to use Vite, and also I learned how to import custom fonts and classes into Tailwind.

```index.css
@import url("https://fonts.googleapis.com/css2?family=Outfit:wght@100..900&display=swap");

@import "tailwindcss";

@theme {
    --font-fm-outfit: Outfit, "sans-serif";
    --color-fm-slate-300: hsl(212, 45%, 89%);
    --color-fm-slate-500: hsl(216, 15%, 48%);
    --color-fm-slate-900: hsl(218, 44%, 22%);
}

```
